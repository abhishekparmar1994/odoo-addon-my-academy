My Academy Webmyne

Turn Odoo eLearning into a powerful monetization engine. Enforce lesson completion gating, sell PDF certificates, and process checkouts via Stripe.

Odoo 19.0 Ready Stripe Integrated eLearning & eCommerce

Elevate Your Online Academy

This module fills critical functional gaps in Odoo's native eLearning by combining structural course gates with commerce-optimized certificate checkouts.

🔒

Lesson Gating

Blocks certification quiz access and PDF downloads until the student has marked every prerequisite lesson as completed.

💳

Stripe Checkouts

Seamlessly links survey certifications to purchasable product variants. Automatically handles checkout redirections via eCommerce.

🎨

Elite UI Styling

Replaces default Odoo error blocks with stunning, custom-coded glassmorphism pages. Styled with premium gradients and trust badges.

How It Works under the Hood

1
Course Gates Check

Odoo queries lesson completions. If missing, it locks the certification quiz and displays a modern warning page listing all uncompleted lessons.

2
Configurable Certificate Pricing

Easily set if a certificate is Free or Paid. Paid certificates require Odoo eCommerce checkout before the PDF download is permitted.

3
Fallback Integrity

Automatically assigns missing certificate products on installation and contains intelligent layout fallbacks for guest certificates (no blank company logos!).

4
Stripe Fallbacks

Recognizes confirmed sales orders as well as draft transactions verified via secure Stripe checkout providers.

Quick Installation

  1. Extract the `my_academy_modifications` folder and place it into your repository root directory.
  2. Add the path to your Odoo `addons_path` in your configuration (`odoo.conf`).
  3. Enable **Developer Mode** under Settings, head to **Apps**, and click **Update Apps List**.
  4. Search for `my_academy_modifications` and click **Install**.
